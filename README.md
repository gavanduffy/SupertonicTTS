# SupertonicTTS
Run [SupertonicTTS](https://github.com/supertone-inc/supertonic) models locally on your iOS device with high performance inference.
You first need to download the orginal models from [HuggingFace](https://huggingface.co/Supertone/supertonic/tree/main/onnx) and drop them into ``/Resources`` dictionary.
